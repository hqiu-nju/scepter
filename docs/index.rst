SCEPTer Documentation
=====================

SCEPTer, *Simulating Constellation Emission Patterns for Telescopes*, is a Python
toolkit for modelling satellite constellation emissions and estimating their
impact on radio astronomy observations. The package combines sky tessellation,
satellite propagation, antenna response models, interferometric geometry, and
visualisation utilities for EPFD and RFI studies.

This documentation is organised as a practical guide to the repository: how to
set up the project, how the main simulation workflows fit together, and where to
find the key modules inside the package.
